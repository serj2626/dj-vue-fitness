### Инструменты

#### Backend

- Python
- Django
- Django Rest Framework
- CK-Editor
- PostgreSQL
- Celery
- Redis
- Docker

#### Frontend

- Vue.js
- Typescript
- Pinia
- Tailwind
- Swiper
- Gsap
- Axios
- PrimeVue
- Flowbite
- Docker

---

### Функционал

- Регистрация
- JWT авторизация
- Просмотр постов
- Детальная информация о тренере
- Оставление отзыва и рейтинга тренеру
- Запись на персональную тренировку
- Покупка абонемента
- Подписка на новости

---

### Скачать проект

> через github

    git clone https://github.com/serj2626/dj-vue-fitness

> через docker

    docker pull serj2626/dj-vue-fitness

---

### Команды для запуска сайта

> Backend

```bash
python manage.py runserver
```

> Frontend

```bash
npm run dev
```
